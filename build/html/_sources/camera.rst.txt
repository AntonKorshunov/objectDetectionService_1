Camera
======

Описание класса Camera
----------------------

Класс Camera отвечает за захват видеопотока с использованием библиотеки OpenCV. Предоставляет функционал для чтения и обработки видеопотока в реальном времени.

Методы класса
-------------

- ``__init__(self, rtsp_link)``:
  Конструктор класса, принимающий RTSP-ссылку и инициализирующий объект захвата видеопотока.

- ``rtsp_cam_buffer(self, capture)``:
  Метод, работающий в отдельном потоке, для непрерывного чтения кадров из видеопотока.

- ``getFrame(self)``:
  Метод для получения последнего захваченного кадра из потока.
